# AI驱动的健康管理平台

一个基于AI技术的健康管理应用，提供个性化的健康方案和实时数据分析。

## 项目功能

- 用户认证与个人信息管理
- 健康指标监测与分析
- 个性化饮食计划生成
- 运动建议与训练计划
- 心理健康陪伴

## 技术栈

- 前端：Vue.js, TypeScript
- 样式：CSS
- 工具：Git

## 本地开发

1. 克隆项目
   ```bash
   git clone https://github.com/shi-an/health-guardian
   ```

2. 安装依赖
   ```bash
   npm install
   ```

3. 启动开发服务器
   ```bash
   npm run dev
   ```

## GitHub 配置指南

要将此项目推送到GitHub，请按照以下步骤操作：

1. 在GitHub上创建新仓库：
   - 登录GitHub并点击右上角的加号
   - 选择"New repository"
   - 填写仓库名称（建议使用与本地项目相同的名称）
   - 选择公开或私有
   - 点击"Create repository"

2. 复制GitHub仓库URL

3. 在本地项目目录中配置远程仓库：
   ```bash
   git remote add origin https://github.com/shi-an/health-guardian
   ```

4. 推送到GitHub：
   ```bash
   git push -u origin master
   ```
